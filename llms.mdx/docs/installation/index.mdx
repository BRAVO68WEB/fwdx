# Installation (/docs/installation)



Installation [#installation]

`fwdx` has two sides:

* a public server
* one or more tunnel clients

Requirements [#requirements]

* Go `1.25+` to build from source
* a Linux server with a public IP for the server side
* a domain such as `tunnel.example.com`
* TLS certificates for the public hostname

Paths [#paths]

* Server binary: `/usr/local/bin/fwdx`
* Client config: `~/.fwdx/client.json`
* Detached runtime state: `~/.fwdx/run`
